# WayTure

## Descripción del Proyecto

WayTure es una página web diseñada como una plataforma interactiva que permite a los usuarios explorar y gestionar información a través de una interfaz web moderna. El proyecto implementa tecnologías fundamentales del desarrollo web como **HTML, CSS y JavaScript**, organizadas de forma estructurada para facilitar la navegación, el mantenimiento del código y la escalabilidad del sistema.

La aplicación incluye funcionalidades de autenticación básica mediante páginas de **inicio de sesión** y **registro**, además de una interfaz principal accesible desde el archivo `index.html`.

El proyecto se encuentra desplegado mediante **GitHub Pages**, lo que permite acceder a la aplicación directamente desde un navegador web.

**Acceso a la página:**  
https://skizm07.github.io/WayTure/
